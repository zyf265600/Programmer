# Go Package

## Go 包（Package）

Go 程序是由多个**包**链接而成的。一个包由一个或多个源文件（`.go`）组成。这些文件共同声明属于该包的常量、类型、变量和函数。在同一个包的所有源文件中，这些成员都是相互可见、可直接访问的。此外，包内的元素可以被**导出**，供其他包使用。

### 源文件组织结构

每个源文件由以下三部分按顺序组成：

1. **Package 子句**：定义该文件所属的包名。
2. **导入声明（可选）**：声明该文件所依赖的其他包。
3. **顶层声明（可选）**：声明常量、类型、变量和函数。

```go
package xxx // Package 子句

import (...) // 导入声明

const ...    // 顶层声明
var ...
type ...
func ...
```

### Package 子句

Package 子句位于每个源文件的开头。

- **语法规范**：`PackageClause = "package" PackageName .`
- **限制**：包名（PackageName）必须是有效的标识符，且不能是空标识符（`_`）。

例如：`package math`

一组拥有相同包名的文件共同构成该包的实现。在大多数实现（如官方编译器）中，同一个包的所有源文件必须位于**同一个目录**下。

### 导入声明（Import Declaration）

导入声明指明当前文件依赖于被导入包的功能，并允许访问该包中**已导出**的标识符。导入声明会为导出的标识符定义一个访问前缀（本地名称），并指定被导入包的路径。

- **语法规范 EBNF 范式**：

  `ImportDecl = "import" ( ImportSpec | "(" { ImportSpec ";" } ")" ) .`

  `ImportSpec = [ "." | PackageName ] ImportPath .`

  **ImportSpec（导入项）**：可以是一个单独的导入，也可以是用圆括号包裹的组合导入（Import block）。

  ````go
  // 分组导入，官方推荐做法，清晰整洁
  import (
      "fmt" // 在Go的安装目录下
      "math"
  )
  ````

  **PackageName（包名别名）**：这是一个**可选**的标识符。

  - 如果省略：默认使用被导入包在其源文件中定义的 `package` 名称。
  - 如果显式指定：则在当前文件中使用该名称作为访问前缀。

  **ImportPath（导入路径）**：必须是一个**字符串字面量**（双引号包裹），它是编译器寻找包物理位置的唯一线索。

#### 导入方式与访问规则

1. **默认导入**：如果不指定本地包名，则使用被导入包在 `package` 子句中定义的原始包名。
2. **重命名导入**：通过显式指定 `PackageName` 来定义本地别名。
3. **点导入（`.`）**：如果使用句点代替名称，则该包所有导出的标识符将直接注入到当前文件的文件块中，访问时无需前缀（慎用，易导致命名冲突）。
4. **匿名导入（`_`）**：如果仅为了触发包的**初始化（副作用）**而不直接使用其代码，请使用空标识符作为包名。

**示例：假设导入路径为 `"lib/math"` 的包导出了函数 `Sin`**

| **导入声明**          | **访问方式** | **说明**               |
| --------------------- | ------------ | ---------------------- |
| `import "lib/math"`   | `math.Sin`   | 默认方式               |
| `import m "lib/math"` | `m.Sin`      | 使用别名 `m`           |
| `import . "lib/math"` | `Sin`        | 直接访问（无前缀）     |
| `import _ "lib/math"` | (不可访问)   | 仅执行包的 `init` 函数 |

#### 实现限制

- **依赖限制**：包不能直接或间接导入自身。禁止导入包后而不使用其任何导出标识符（匿名导入除外）。
- **字符限制**：导入路径通常是编译后包文件路径的子字符串。出于安全和跨平台考虑，编译器通常限制路径仅能使用 Unicode 的图形字符，并排除 `!"#$%&'()*,:;<=>?[\]^{|}` 等特殊字符。

### 示例：并发素数筛

下面是一个完整的 Go 包示例，展示了如何通过 goroutine 和 channel 实现并发的素数过滤。

```go
package main

import "fmt"

// generate 向通道 ch 发送序列 2, 3, 4, ...
func generate(ch chan<- int) {
	for i := 2; ; i++ {
		ch <- i
	}
}

// filter 从 src 通道读取数据，过滤掉能被 prime 整除的数，
// 并将剩余数据发送到 dst 通道。
func filter(src <-chan int, dst chan<- int, prime int) {
	for i := range src {
		if i%prime != 0 {
			dst <- i
		}
	}
}

// sieve（素数筛）：通过动态串联过滤进程来提取素数。
func sieve() {
	ch := make(chan int) // 初始通道
	go generate(ch)      // 启动生成器
	for {
		prime := <-ch      // 从通道中获取当前的第一个数（必为素数）
		fmt.Println(prime) // 打印素数
		
		ch1 := make(chan int)
		go filter(ch, ch1, prime) // 基于当前素数建立新的过滤器
		ch = ch1                  // 更新通道，进入下一轮迭代
	}
}

func main() {
	sieve()
}
```
